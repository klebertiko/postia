# **App Name**: InstaBoost AI

## Core Features:

- Legenda Generator: Gere legendas cativantes com limite de caracteres, considerando as informações inseridas sobre a imagem.
- Hashtag Suggestor: Sugira hashtags relevantes com base na descrição da imagem e no tema da postagem. Limite de sugestões para evitar excesso.
- Gemini Nano Prompt Generator: Crie um prompt otimizado para o modelo Gemini Nano para geração de imagem a partir da descrição fornecida. It serves as a tool, leveraging information on current Instagram trends and the user’s existing description to generate creative suggestions.
- Formulário de Entrada: Formulário intuitivo para inserir descrição da imagem, tema da postagem e outras informações relevantes para os geradores.
- Visualização de Resultado: Mostre os resultados (legenda, hashtags e prompt) de forma organizada para facilitar a cópia e o uso no Instagram.

## Style Guidelines:

- Primary color: Rich purple (#6A459E), evoking creativity and luxury, important for Instagram aesthetics.
- Background color: Very light purple (#F0ECF4), providing a subtle and clean base.
- Accent color: Vibrant pink (#E91E63), drawing attention to important actions and highlights.
- Headline font: 'Poppins', sans-serif.
- Body font: 'PT Sans', sans-serif
- Ícones minimalistas e modernos, alinhados com a identidade visual do Instagram.
- Layout limpo e organizado, priorizando a usabilidade e a clareza das informações.
- Animações sutis e agradáveis ao gerar novos conteúdos, melhorando a experiência do usuário.
